# ⚡ 閃電水電行 (Lightning Electric Shop) ⚡


> **「嘿！大老闆，今天的報修單也爆滿了，準備好迎接挑戰了嗎？」**

這是一款專為國小 3-4 年級學生設計的 **Web 互動式科學遊戲**。玩家將化身為「閃電水電行」的老闆，透過處理鄰居們各式各樣的電器維修需求，學習並應用「電池串並聯」與「燈泡串並聯」的基礎電路科學邏輯。

## 🌟 遊戲特色

* **隨機任務系統**：從 15 位個性鮮明的角色中隨機抽選 3 位客戶，每次遊玩都有不同驚喜。
* **多樣化評價**：根據玩家的電路配置，給予 1 星（客訴）到 3 星（完美）的即時對話回饋。
* **隱藏魔王關卡**：完成基礎任務後，可選擇是否挑戰「演唱會歌手」的最難電路配置。
* **科學經營稱號**：結算時根據總分給予稱號，包含「技術超強」、「偶有小失誤」或「技術待加強」。
* **知識內化**：遊戲結算畫面會回顧每題的「維修邏輯」，幫助玩家加強科學記憶。

## 🧪 科學邏輯對照表

在遊戲中，玩家需要掌握以下核心概念：

| 維修需求 | 電池配置 | 燈泡配置 | 科學原理 |
| :--- | :--- | :--- | :--- |
| **電力持久** | 並聯 | - | 分擔電流，延長使用時間 |
| **爆發動力** | 串聯 | - | 疊加電壓，增加功率 |
| **亮度最強** | 串聯 | 並聯 | 最大推力 + 獨立路徑發光 |
| **柔和光線** | - | 串聯 | 分攤電壓，降低亮度 |
| **互不干擾** | - | 並聯 | 獨立電路，壞一顆另一顆仍亮 |

## 🛠️ 技術架構

* **Frontend**: HTML5, CSS3 (Flexbox/Animation), JavaScript
* **Backend**: Google Apps Script (GAS)
* **API Integration**: Google Apps Script `doGet` & `google.script.run`

## 📂 檔案結構說明

* `Code.gs`: 負責後端邏輯判斷、15 位角色的評價資料庫及魔王關判定。
* `index.html`: 包含遊戲介面設計、狀態機切換（State Machine）及前端互動邏輯。
